# Política de Privacidade - OS Flash

Esta política de privacidade descreve como o aplicativo **OS Flash**, coleta, usa e protege as informações dos usuários.

## 1. Coleta de Informações
O OS Flash coleta informações de forma limitada para garantir o funcionamento das ordens de serviço e a monetização do app:
- **Fotos e Câmera:** O app solicita acesso à câmera e galeria para anexar fotos às ordens de serviço. Essas fotos são armazenadas localmente no dispositivo.
- **Armazenamento:** Utilizamos o armazenamento local do dispositivo para salvar os dados das ordens de serviço.
- **Identificadores de Dispositivo:** Coletamos IDs técnicos para fins de publicidade e análise de assinaturas.

## 2. Serviços de Terceiros
Para oferecer funcionalidades extras e manter o app gratuito, utilizamos os seguintes serviços que possuem suas próprias políticas de privacidade:
- **Google AdMob:** Para exibição de anúncios. O AdMob pode coletar dados como o ID de publicidade do seu dispositivo para personalizar anúncios.
- **RevenueCat:** Para gerenciar as assinaturas Premium de R$ 5,00. O RevenueCat processa informações de compra, mas não temos acesso aos seus dados de cartão de crédito (gerenciados pelo Google Play).

## 3. Segurança
Temos o compromisso de garantir que suas informações estejam seguras. Como o app funciona prioritariamente de forma local, os dados da sua empresa e clientes permanecem no seu dispositivo.

## 4. Retenção e Exclusão de Dados
Você pode excluir seus dados a qualquer momento limpando os dados do aplicativo nas configurações do Android ou desinstalando o app.

## 5. Alterações nesta Política
Podemos atualizar nossa Política de Privacidade ocasionalmente. Recomendamos que você revise esta página periodicamente.
